  • Download the Project Template from Visual Studio Market Place
  • Run the downloaded VSIX file
  • Open the Visual Studio
  • Click on create the project
  • Select the Project Template of Clean Architecture of AspNetCore and Goto Next
  • Enter the Project Name and tick the option of 'Place Solution and Project in the same directory'
  • Open Command Prompt
  • Install the dotnet CLI of AspNetCore Clean Architecture
  • Run the command for creating new project : dotnet new rxwebapi --name HumanResourceApplication
